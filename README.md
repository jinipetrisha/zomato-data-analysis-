# zomato-data-analysis-
# 🍽️ Zomato Data Analysis Project

## 📌 Project Overview
This project analyzes Zomato restaurant data to discover useful insights about restaurants, customer preferences, ratings, and location trends. Using Python data analysis libraries, we explore patterns that help understand food trends and restaurant performance.

The goal of this project is to perform **data cleaning, exploration, and visualization** to extract meaningful insights from the dataset.

---

## 🎯 Objectives
- Understand restaurant distribution across locations
- Analyze restaurant ratings and votes
- Identify popular cuisines
- Discover price range patterns
- Visualize customer preferences

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset
The dataset contains information about restaurants such as:

- Restaurant Name
- Location
- Cuisine Type
- Ratings
- Number of Votes
- Price Range
- Online Order Availability

---
## 📊 Project Workflow
### 1️⃣ Data Collection
Import the Zomato dataset and load it into a Pandas DataFrame.
### 2️⃣ Data Cleaning
- Handle missing values
- Remove duplicates
- Convert data types
- Filter unnecessary columns
### 3️⃣ Exploratory Data Analysis (EDA)
Analyze data patterns such as:
- Most popular restaurant locations
- Highest rated restaurants
- Cuisine popularity
- Rating distribution
### 4️⃣ Data Visualization
Use graphs and charts to visualize insights:
- Rating distribution
- Restaurant count by location
- Cuisine popularity
- Votes vs Ratings
Libraries used:
- Matplotlib
- Seaborn
---
## 📈 Key Insights
Some insights discovered from the analysis:
- Certain locations have significantly more restaurants.
- Popular cuisines attract higher ratings.
- Restaurants with more votes often have better ratings.
- Price range impacts restaurant popularity.
